Este exemplo representa uma aplicação genérica do padrão de desenho ObjectPool.
Neste exemplo mostra a utilização de 2 "pools" criadas apartir de uma classe que recebe um objeto/classe genérico, dando mais flexibilidade a aplicação, servindo como solução para qualquer problema em que consideramos necessário aplicar este padrão.

A classe Client representa as associações entre os clientes e as "pools", em que irão interágir requisitando objetos e os devolvendo
