Este exemplo representa um sistema de uma loja de aluguer de carros e bicicletas para crianças.
A loja contem 2 "pools" na qual irão conter um stock limite ( capacidade maxima), quando a loja é iniciada é atribuido um ID unico a cada objeto, entre outros atributos. 

O cliente interage com a loja ao requisitar uma bicicleta ou a devolve-la, sem ter conhecimento de onde é guardado os objetos na loja.
